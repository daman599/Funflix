Funflix 

It is MERN tech-stack based web app which tells where to watch the searched movie, which platforms it is available on.

Things I implemented here :-

# Fronted

- TailwindCss
- ReactJs
- Motion
- Api Integration

# Backend

- Google Oauth (manually)
- Express for defining api endpoints
- Middleware
- Token based authentication
- MongoDB database
- Input validation using Zod
